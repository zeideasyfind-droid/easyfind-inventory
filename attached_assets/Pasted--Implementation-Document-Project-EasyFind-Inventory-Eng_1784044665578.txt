## Objective



Build a production-ready web application where a broker pastes a property listing URL and receives structured inventory saved into Google Sheets.



---



## User Flow



1. User opens the webpage.

2. Pastes a Housing.com property URL.

3. Clicks **Extract Property**.

4. Backend calls Firecrawl.

5. Firecrawl extracts the webpage.

6. LLM converts raw HTML into structured JSON.

7. Normalize data.

8. Check duplicates.

9. Save JSON archive.

10. Append inventory to Google Sheets.

11. Return success message.

## JSON Archive



Every extraction must be archived before writing to Google Sheets.



Store JSON under:



```

archives/YYYY/MM/DD/

```



Filename:



```

property_<timestamp>.json

```



Use Google Drive (preferred) or local storage initially.



Implement in:



```

backend/services/google_drive.py

```
